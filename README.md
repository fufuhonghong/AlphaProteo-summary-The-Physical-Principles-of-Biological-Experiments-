# AlphaProteo-summary-The-Physical-Principles-of-Biological-Experiments
AlphaProteo Reading Notes

This repository contains my personal reading notes on the AlphaProteo paper.

I am a physics student with a strong interest in biology and the intersection between AI and biological systems. However, my background in experimental biology is still limited.

Because of this, I wrote this document as a way to help myself understand the experimental workflow from a more physical and intuitive perspective.


📌 Note

This is not an official summary or review, but a personal set of structured notes.

The focus is on:
	•	building intuition for biological experiments using physical concepts
	•	understanding experimental pipelines (expression, binding, detection)
	•	bridging AI-designed proteins with experimental validation
	•	binding affinity (Kd) analysis
	•	structural validation and interface interpretation


📄 Reference

AlphaProteo paper: https://arxiv.org/abs/2409.08022
