# AlphaProteo-summary-The-Physical-Principles-of-Biological-Experiments
🧬 AlphaProteo Reading Notes

This repository contains my personal reading notes on the AlphaProteo paper.



👤 About Me

I am a physics student with a strong interest in biology and the intersection between AI and biological systems.

My background in experimental biology is still limited, so this document is my attempt to understand biological experiments through a more physical and intuitive lens.



🎯 Focus of These Notes

These notes are not an official review, but a structured personal study record.

They focus on:
	•	building intuition for biological experiments using physics-based thinking
	•	understanding experimental pipelines (expression → binding → detection)
	•	connecting AI-designed proteins with experimental validation
	•	interpreting binding affinity (Kd) from a physical perspective
	•	understanding structural/interface-level validation




📄 Reference
	•	AlphaProteo paper: https://arxiv.org/abs/2409.08022
